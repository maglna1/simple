Python 是一种广泛使用的高级编程语言，由 Guido van Rossum 于 1991 年首次发布。
它强调代码的可读性和简洁的语法（特别是使用缩进划分代码块，而不是括号或关键字）。
Python 支持多种编程范式，包括面向对象、命令式、函数式和过程式编程。

LangChain 是一个用于构建大语言模型（LLM）应用的开发框架。
它提供了模块化的组件（如 Prompt 模板、向量存储、Agent），让开发者能够把 LLM 与外部数据、工具、工作流组合起来。
LangChain 的核心思想是"链式组合"——把多个可执行组件用统一的接口串起来。

RAG（Retrieval-Augmented Generation，检索增强生成）是一种结合信息检索与文本生成的技术。
其核心思路是：在让 LLM 回答之前，先从一个大型文档库中检索出与问题相关的片段，
然后把这些片段连同问题一起作为 prompt 喂给 LLM。这样可以让 LLM 的回答基于最新的、特定的领域知识，
而不仅仅依赖训练时学到的内容，从而显著降低"幻觉"现象。

向量数据库是 RAG 系统的关键组件之一。它把文本转换为高维向量（通过 Embedding 模型），
存储起来，并在查询时通过相似度检索找出与问题最相关的文档片段。
常见的向量库包括 Chroma、FAISS、Pinecone、Weaviate 等。
